Box Breathing 4min

Minimalist 4-minute guided box breathing exercise.

Overview

Box Breathing 4min is a calm, distraction-free breathing exercise app. Follow the glowing dot around a square: breathe in, hold, breathe out, hold. The 4-minute session starts at a 3-second rhythm and gently transitions to 4 seconds halfway through, easing you into deeper calm. No accounts, no ads, no tracking, no internet required.

Features

  • Single-screen, distraction-free interface
  • Animated 60fps guide tracing the square
  • Progressive rhythm: 3s phase ramping to 4s
  • Bilingual prompts (English / French)
  • Screen stays awake during the session
  • Works fully offline; no permissions, no telemetry

Screenshots

Build & install

The project is an Expo / React Native app (managed workflow).

npm install
npx expo prebuild --platform android --clean
cd android && ./gradlew bundleRelease

For iOS, see DEPLOY.md for the full Codemagic pipeline (App Store Connect + TestFlight).
